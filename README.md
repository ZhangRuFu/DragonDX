# Why DirectX
有过OpenGL的基础，虽然想重新看一下OpenGL，但是最终选择了学习DirectX 11，原因如下：
1. 学习一门新的图形API
2. 通过比较两者的差异更好的掌握OpenGL和DirectX
3. 使用Windows + Visual Studio开发学习，使用DirectX 11更方便
